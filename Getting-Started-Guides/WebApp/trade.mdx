The **Trade** page is where you can **swap one token for another** instantly using SaucerSwap's decentralized exchange.\
You can select:

- The token you’re swapping **from**
- The token you’re swapping **to**
- Adjust settings like **slippage tolerance**

This is the most used page for everyday crypto trading.
