REST APIs for tokens on SaucerSwap

Access the SaucerSwap API using the following base URLs for testnet and mainnet environments:

| Hedera Network | Base URL                                                                              |
| -------------- | ------------------------------------------------------------------------------------- |
| Mainnet        | [https://api.saucerswap.finance/](https://api.saucerswap.finance/v2/pools/)           |
| Testnet        | [https://test-api.saucerswap.finance/](https://test-api.saucerswap.finance/v2/pools/) |
| Previewnet     | *Not supported*                                                                       |

### Get historical candlestick data for token by id

get

/tokens/prices/*\{tokenId}*

Test it

Path parameters

tokenIdstringrequired

Example: `0.0.731861`

Query parameters

fromintegerrequired

From Unix seconds

Example: `1697600000`

tointegerrequired

To Unix seconds

Example: `1697610000`

intervalstring · enumrequired

Example: `HOUR`

Options: `FIVEMIN`, `HOUR`, `DAY`, `WEEK`

Responses

200

Successful response

application/json

<Tabs />

```
curl -L \
  --url '/tokens/prices/{tokenId}?from=1697600000&to=1697610000&interval=HOUR'
```
